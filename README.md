Abastece Bot - Inteligência Artificial 🤖🏥

O Abastece Bot é um assistente virtual inteligente desenvolvido para a startup CasaAbastecida. O foco do projeto é automatizar o atendimento ao cliente, ajudando-o a entender prazos, entregas e facilitar a gestão do consumo doméstico através de um diálogo fluido e personalizado.

🎯 Objetivo do Projeto

Responder às principais perguntas do dia a dia da operação da CasaAbastecida, proporcionando conveniência e eficiência na reposição de itens essenciais sem intervenção manual do usuário.



🧠 Tecnologias Utilizadas

IBM Watson Assistant: Plataforma utilizada para a criação da inteligência, gerenciamento de intenções e fluxos de diálogo.  

JSON: Formato utilizado para exportação e integração da Skill treinada.



🛠️ Funcionalidades e Personalizações

Seguindo as diretrizes do Checkpoint, o bot conta com:  


Intenções Robustas: Cada intenção foi treinada com, no mínimo, 7 exemplos de frases para maior precisão.  


Entidades Ricas: As entidades possuem ao menos 4 sinônimos cada, facilitando o reconhecimento de termos variados.  


Bem-Vindo Dinâmico: O nó de saudação apresenta 3 variações de mensagens, cada uma acompanhada de uma imagem diferente para uma interface mais rica.

📁 Estrutura de Entrega

Skill JSON: Arquivo exportado com todo o treinamento da IA.  


Link de Preview: Acesso direto para testar o chatbot em tempo real.  


Vídeo Demonstrativo: Demonstração prática das personalizações e funcionamento geral
